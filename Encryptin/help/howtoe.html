How to start?

Using Encryptin is easy: all you need to do is REGISTER
with Google Chrome using a USERNAME AND PASSWORD
.


Once registered, you can LOGIN
. Once registered, you can login and decide if you want your credentials to be remembered. If you don't want your credentials to be remembered, you will be asked to enter them for each OPERATION
.

Logging in, you can access the start MENU
from where you can perform other operations explained in this help pages.

To start using the encryption system Encryptin we first have to have an encryption key.
