Local Files

Encryptin allows you to encrypt and decrypt files on your computer.
This feature can be accessed through the MAIN MENU
,clicking on LocalFile
To encrypt a file, simply press Encrypt local file and select the file you want to encrypt.

This will create an encrypted copy of the selected file with the extension ".encrypted" that will be saved in chosen directory.

Clicking on Decrypt local file, you can decrypt a ".encrypted" file.

The decrypted file will be saved in the default Download directory of Google Chrome.

(Default on Windows = C:\Users\USER_NAME\Downloads)
