File Locali

Encryptin permette di cifrare e decifrare file in locale.
Attraverso il MENU' PRINCIPALE
, premendo Local Files, si accederà al MENU
per manipolare i file locali, sarà possibile premendo Encrypt local file cifrare un file.

Verrà creata una copia cifrata ".encrypted" del file selezionato che verrà salvata nella directory in cui si è scelto di far scaricare i file a Google Chrome.

Premendo invece Decrypt local file, sarà possibile decifrare un file precedentemente cifrato ".encrypted".

Il file decifrato in questo modo verrà anche esso salvato nella directory in cui Google Chrome salva i file scaricati.

(Default su Windows = C:\Users\NOME_UTENTE\Download)
